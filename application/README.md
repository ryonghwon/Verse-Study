# Web Application
- Restful API : **Verse** 사용자 / 관리자에서 사용되는 Restful API 구축
- Admin : **Verse** 관리자용 웹앱 제작
- Client : **Verse** 사용자용 웹앱 제작


## Installation

```bash
yarn install
```

## Running the Application

- Development Mode

```bash
# development mode
yarn dev
```

- Production Mode

```bash
# project build
yarn build

# production mode
yarn start
```

- ES Lint Check

```bash
# lint check
yarn lint

# lint check & fix
yarn lint:fix
```
